Acquia Search module
================================================================================

Provided integration between your Drupal site and Acquia's hosted search
service, Acquia Search [1]. Requires Search API Solr module.

[1] https://docs.acquia.com/acquia-search/

This module is no longer being used. It exists for stub purposes only.
You should uninstall this module immediately and use Acquia Search Solr [2] instead.

[2] https://www.drupal.org/project/acquia_search_solr
