INTRODUCTION
------------

Simple Comment Notify is a lightweight tool to send notification e-mails to
admins/moderators when user create comment.

 * For a full description of the module, visit the project page:
   https://drupal.org/project/scn

 * To submit bug reports and feature suggestions, or to track changes:
   https://drupal.org/project/issues/scn


REQUIREMENTS
------------

comment (drupal core)


INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. Visit:
   https://www.drupal.org/node/1897420 for further information.


CONFIGURATION
-------------

    1. Navigate to Administration > Extend and enable the module.
    2. Navigate to Administration > Configure > System > Simple Comment Notify 
       to configure the module.
    3. Save configuration.
